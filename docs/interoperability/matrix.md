# Interoperability with Matrix API

Matrix is a protocol for secure, decentralized communication. It's the protocol used by [Tchap](https://www.tchap.gouv.fr/), another application of [La Suite](https://lasuite.numerique.gouv.fr/).

## Purpose

Webhooks using this protocol means to automatize the invitation and removal of accounts to Matrix chat rooms upon editing the groups you manage in People.

> [!NOTE]   
> The first version of this feature requires inviting a bot and giving it moderation permission to your Matrix room. A second version of the feature, in which invitations/removal will be called using your own Matrix/ProConnect account will be delivered later this year.
